3 基于PVLM的 Few-shot 多模态情感分析

  在本节中，首先阐述本文的任务。然后，详细介绍了提出的用于 FMSA (Few-shot Multi-Modal Sentiment Analysis) 的 PVLM。以往方法的简化结构 和 PVLM 的结构 Figure 1 所示。

3.1 任务定义

sentence-level MSA：   给定一个多模态样本 S，它包含一个句子 X=x1,x2,···,xn 和一个相关联的图像 Img，其中 n 表示文本中的字数。在此基础上，我们的目标是识别整个文本所表达的情感极性 y∈{positive,neutral,negative,···} 或 情感类别 y∈{happy,angry,sad,···}。

aspect-level MSA：   给定一个多模态样本 S，除了一个句子 X 和一个相关联的图像 Img，在 X 中还存在一个短语或词作为 aspect A={a1,a2,···，ak}，其中 k 表示短语词的个数。在此基础上，我们的目标是识别关于 A 的情绪极性 y∈{positive,neutral,negative}。

  在之前研究的基础上，我们对每个任务的训练集 Dtrain，每类只假设 K 个训练实例（例如 12个），使得样本总数 Ktotal=K×|Y|，其中 Y 是标签空间 (label space)。

3.2 用于 FMSA 的基于 prompt 的视觉感知语言模型

  为了更好地描述，作者首先简单地介绍了以往 BERT 和视觉结合的 tuning 方法，然后介绍了本文的 PVLM 方法，通过视觉感知 prompt tuning 用于 FMSA。

BERT 和视觉结合的 previous tuning：   在文本场景中，传统的方法，即 previous tuning，通常将输入 X 转换为一个 token 序列 [CLS]X[SEP]，并将其传递给 BERT，得到隐向量序列 {hi∈ℝd}。然后将 h[CLS] ([CLS] 隐藏向量) 传入一个 softmax 层，Softmax(Wph[CLS])，以获得情感标签。这可以参考 Figure 1 (b)。

  在多模态场景中，文本模态和 aspect 项通常由 BERT 编码来得到序列 Hx={hi∈ℝdx} 和 Ha={hj∈ℝdx}，视觉模态由 ResNet 编码得到离散特征矩阵 He={ei∈ℝde}。然后，跨模态注意被用来执行 aspect 和视觉得交互作用 E=CM(Ha,Hc)，其中 CM 表示跨模态注意函数 (cross-modal attention function)。随后，E 和 Hx 被 concat 并传入多头注意力 (multi-head attention) 层，通过 pooling 产生混合向量 h~ 。最后，h~ 被输入到 Softmax(Wph~)，获得情感标签。这可以参考 Figure 1 (c)。

  请注意，对于上述过程中的任何多模态任务，新参数的数量可能很大。例如，对于其特定标签空间的跨模态注意力和分类层，这使得从少量标注数据中学习 FMSA 相当困难。

基于 prompt 的语言建模 和 few-shot 学习：

  基于prompt 的语言建模被认为是 NLP 的第四范式。由于预训练和下游任务的形式一致，它可以在许多基于文本的任务中工作，并且仅仅是 few-shot 甚至 zero-shot 的：文本分类 (AutoPrompt: Eliciting Knowledge from Language Models with Automatically Generated Prompts)、文本回归 (Making Pre-trained Language Models Better Few-shot Learners) 和文本生成(Prefix-Tuning: Optimizing Continuous Prompts for Generation)。然后，它们只能处理文本模态。直到最近，据作者所知，只有一个关注多模态数据的 few-shot 研究 (Multimodal Few-Shot Learning with Frozen Language Models) 主要与本文的 PVLM 方法有关。然而，这项研究依赖于多模态数据的大规模预训练。除此之外，它还可用于视觉问答 (VQA)。

  与上述研究不同的是，PVLM 可以处理多模态输入，而无需大规模多模态预训练，并充分利用了视觉集成的掩码语言模型用于FMSA。
